# Weather App

## How to Run

```shell
$ cd frontend && npm install
$ cd ..
$ cd backend && npm install && npm start
$ cd ..
$ cd frontend && npm start
```

## Required environment variables

for backend

- OPENAPI_KEY
- API_NINJA_KEY
- TWITTER_BEARER_TOKEN
